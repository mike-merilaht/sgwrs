# sgwrs

/sɡʊrs/

noun: (welsh) conversation

A basic WebSocket server with a chat implementation (no history)

## spec

[rfc6455](https://datatracker.ietf.org/doc/html/rfc6455)

## Why?

To learn how WebSockets work, and learn golang.

## Developing

### nix-shell

If you want, you can use nix-shell here to set up the environment

```bash
nix-shell
```

### Run

```bash
go run main.go
```


### Tests

```bash
go test ./...
```

## Should I use it?

At your own risk.
